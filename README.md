🏝️ 신비한 수학의 섬, 구구랜드!

초등학교 2학년 친구들을 위한 즐거운 구구단 게임입니다.

🎮 게임 소개

구구랜드는 신비한 수학의 섬에서 펼쳐지는 구구단 대결 게임입니다.
귀여운 캐릭터 파트너와 함께 문제를 풀고, 점수를 모아 '솜씨 왕'에 도전해보세요!

✨ 주요 특징

최대 4인 플레이: 친구들과 한 화면에서 같이 할 수 있어요.

귀여운 파트너: 2단부터 9단까지, 각 단을 대표하는 8명의 캐릭터가 있어요.

박진감 넘치는 점수 시스템:

기본 점수: +10점

⚡ 속도왕: 5초 안에 풀면 +5점 추가!

🔥 연속 정답: 3문제 연속 정답부터 +5점 추가!

✨ 파트너 보너스: 내 캐릭터와 같은 단이 나오면 +1점 추가!

솜씨 자랑 (명예의 전당): 점수가 높으면 기록에 남아요.

🕹️ 게임 방법

1P ~ 4P 중 원하는 위치에 이름을 입력하세요.

파트너 선택 버튼을 눌러 마음에 드는 캐릭터를 고르세요.

게임 시작 버튼을 누르면 2분(120초) 동안 게임이 진행됩니다.

자기 차례가 오면 화면 가운데 문제를 보고 정답을 입력하세요.

가장 높은 점수를 얻은 친구가 승리!
